# 選択範囲の簡易要約ブックマークレット

Web ページ上で選択したテキストを Gemini Nanoに要約させ、箇条書きで表示する

要約結果はページ右下にポップアップ表示され、編集・コピーできる。

---

## 機能概要

- Webページ上で選択したテキストをその場で取得

- Chrome 内蔵の Gemini Nano（Summarizer API） を使用
-
- 重要ポイントを箇条書き要約

- 結果は右下のポップアップ UI に表示

- 「コピー」ボタンでクリップボードに保存可能

---

## 例

- [サクラエディタ](https://ja.wikipedia.org/wiki/%E3%82%B5%E3%82%AF%E3%83%A9%E3%82%A8%E3%83%87%E3%82%A3%E3%82%BF)を要約

### 要約結果

- サクラエディタはWindowsで動作する、オープンソースの日本語テキストエディタです。仲谷紀夫氏が1998年から開発を開始し、2000年にソースコードを公開しました。
- 2001年にはSourceForge.netに開発環境を移し、ソフトウェア名を「サクラエディタ」に変更しました。その後、2018年にGitHubへ移転しました。
- クリーム色の背景が標準で設定されており、ソースコード編集に適したキーワード強調表示やアウトライン解析などの機能が利用できます。複数エンコードやGrep、マクロも完備。
- 様々な文字コードに対応しており、Unicode版以降はShift_JISで表現できない文字も扱えるようになりました。初期版では改行コードの制約がありましたが、現在は改善されています。
- 著作権に関する明確なライセンスが確立されておらず、zlibライセンスの適用に向けた議論を経て、2019年3月リリースのバージョンからzlibライセンスで配布されています。
