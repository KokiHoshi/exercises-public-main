## npxを利用する利点

### 1. パッケージをインストールせずに実行できる

npx を使うと、npm パッケージをグローバルインストールすることなく一時的にダウンロードして実行できる。そのため、不要なパッケージを環境に残さずにコマンドラインツールを利用できる。

例

```
npx create-react-app myapp
```

→ create-react-app を事前に npm install -g する必要がない。

### 2. プロジェクトのローカル依存パッケージを簡単に実行できる

npx は node_modules/.bin にあるコマンドを自動的に実行できる。そのため、グローバルにインストールされたものではなく、プロジェクトで指定されたバージョンのツールを確実に使用できる。

例

```
npx eslint .
```

### 3. 必要なときだけツールを利用できる

一度しか使わないツールや短期間だけ必要なツールを、インストールせずにその場で実行できる。このため、開発環境をシンプルに保つことができる。
