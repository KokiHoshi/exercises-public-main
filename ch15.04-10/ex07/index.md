## .png 画像や .jpeg 画像と比べた際の SVG

### メリット

- 拡大・縮小しても画質が劣化しない

  SVG は ベクター形式（数式・座標で描画）であり、画面サイズや解像度に依存せず、どれだけ拡大してもジャギーが発生しない

- ファイルサイズが小さくなる場合が多い

  アイコン、ロゴ、図形など単純な画像では PNG / JPEG より軽量

- CSS や JavaScript で操作できる

  SVG 内の要素に対して色変更、アニメーション、表示制御が可能

- テキストとして扱える

  検索エンジンが内容を理解できる

### デメリット

- 写真や複雑な画像には不向き

  写真のような ピクセル単位の情報が多い画像では、ファイルサイズが大きくなりやすい

- 描画コストが高くなる場合がある

  複雑なパスや要素数の多い SVG は、描画時の負荷が高くなる

- セキュリティ上の注意が必要

  SVG はスクリプトを含められるため、
  不適切に扱うと XSS のリスクがある
