## 開発者ツールで CSS に関して実行できる操作

### CSSプロパティの値の変更

color、background-color、margin、padding、font-size などを直接変更し、レイアウトや配色の確認ができる。元のCSSファイルは変更されず、一時的な変更として扱われる。

### 新しいCSSプロパティを追加

Stylesパネルで要素に対して新しいプロパティを追加でき、デザイン調整に使える。

### ボックスモデルの確認

margin / border / padding / content の関係を図で確認でき、要素間のズレや余白の原因を特定できる。

## 15.4-10.2 のアプリの `body` 要素に対し、元々 HTML および JS 内で利用していなかった Tailwind CSS のクラスを開発者ツールから追加すると変更が反映されない理由

Tailwind CSSはビルド時にソースコードをスキャンし、HTMLやJavaScript内で実際に使用されているクラスに対応する CSS のみを生成するから。そのため、開発者ツールから後から追加したクラスは CSS が生成されておらず反映されない。
該当のクラスを使用したい場合は、HTMLやJavaScriptファイル内にクラス名を記述し、CSSを再生成する必要がある。
